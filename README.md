### FastAPI Demo App

### Endpoints
- / -> return hostname, request header info
- /hostname -> return hostname

### Setup - Def
```markdown
python3 -m venv .venv
source .venv/bin/activate
pip install "fastapi[standard]"
fastapi dev main.py
```
